# AI Workflow Rules

## Approach
Build this landing page incrementally. Context files define the scope. Do not invent sections or behaviors not defined in `project-overview.md`.

## Scoping Rules
- Build one section at a time (Hero -> Features -> Testimonials -> etc.)
- Verify responsiveness (Mobile & Desktop) for each section before moving to the next.

## Handling Missing Requirements
- If a copy/text is missing, use clear placeholder text (e.g., "Insert Headline Here").
- If colors are missing, use the placeholder variables.

## Before Finishing
1. All sections are implemented and responsive.
2. Lighthouse performance is > 90.
3. `progress-tracker.md` is updated.
